事前準備

  • VSCodeの拡張機能「GitHub Pull Requests and Issues」をインストール
  • ALGYAN Webサイトリポジトリ（もしくは他の公開リポジトリ）からフォークして自分のリポジトリを作成

1. GitHubリポジトリをクローン

  • 1-1. VSCodeのコマンドパレットを開く

    Windowsの場合は「Ctrl + Shift + p」、Macの場合は「Cmd + Shift + p」

  • 1-2. Gitで検索し、「Git: Clone」を選択

  • 1-3. GitHubリポジトリのURLを入力

    PCの任意のフォルダを選択。選択した場所にリポジトリがcloneされます。

2. 修正 + コミット

  • 2-1. 修正を加える

    リポジトリ内のファイルに対して適当に修正を加えます。

  • 2-2. 修正したファイルをステージングエリアに追加。

  • 2-3. コミット

    差分を確認、コメントを記入しコミットします。

3. Push

  • 3-1. Push

    cloneの時と同様な手順でコマンドパレットから「Git: push」を選択。

4. プルリクエスト

  • 4-1. アクティビティバーの「GitHub」からプルリクエストを作成

  • 4-2. フォーク元のレポジトリを選択

  • 4-3 修正したブランチを入力してEnterキーを押す

  • カスタムメッセージを選択し、コメントを入力
